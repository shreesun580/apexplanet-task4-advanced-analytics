# ApexPlanet Data Analytics Internship - Task 4

## Advanced Analytics and Predictive Modeling

### Project Overview

This project is a part of the **Data Analytics Internship** offered by **ApexPlanet Software Pvt. Ltd.**

The objective of this task is to perform advanced statistical analysis, time series analysis, customer segmentation, and predictive modeling using Python and machine learning techniques.

---

## Objectives

- Perform descriptive statistical analysis
- Conduct hypothesis testing
- Analyze time series data
- Perform customer segmentation using K-Means Clustering
- Visualize clusters using PCA
- Build a predictive model using Linear Regression
- Evaluate model performance using regression metrics
